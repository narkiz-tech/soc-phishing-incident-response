# Tools & Platforms Used

This project was completed using industry-relevant cybersecurity tools commonly found in Security Operations environments.

---

## IRONSCALES

**Purpose:**
- Phishing detection and classification
- Incident visibility and investigation
- Sender risk and behavioural analysis
- Phishing simulation campaigns
- User interaction and awareness metrics

**Used For:**
- Reviewing phishing incidents
- Validating phishing verdicts
- Analysing sender trust indicators
- Assessing campaign effectiveness

---

## Phishing Simulation Module

**Purpose:**
- Simulate real-world phishing attacks
- Measure user behaviour and risk exposure
- Support security awareness training

**Metrics Analysed:**
- Emails delivered
- Click rates
- Lured users
- Awareness indicators
- Campaign timelines

## Supporting Artefacts

- Dashboard screenshots
- Campaign result graphs
- Sender analysis panels
- Incident overview views
